<h2>Dependencies:</h2>
<ul>
  <li>Python 3.x (specifically 3.4 but should run on any)</li>
  <li>Django 1.8</li> 
  <li>pytz (if you want local time)</li>
  <li><h3>included in repository:</h3></li>
    <li>jQuery 1.11</li>
    <li>Bootstrap 3.x</li>
    <li>moment.js (if you want local time)</li>
    <li>underscore.js</li>
  </ul>
<h2>Getting Started:</h2>
  <ol>
  <li>copy repository into Django Project folder</li>
  <li>add BootstrapCalendar to the installed apps list in settings.py.</li>
  <li>add include Bootstrap urls to your project urls.py.</li>
  <li>start website on a localhost and check at the url you set.</li>
  <li>pray and/or troubleshoot.</li>
  </ol>
